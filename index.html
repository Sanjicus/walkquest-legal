WalkQuest

Politique de confidentialité

Dernière mise à jour : mai 2026

1. Introduction

Cette politique de confidentialité décrit comment l'application WalkQuest, développée par Josselin Sanjuan, collecte, utilise et protège vos informations personnelles.

En utilisant WalkQuest, vous acceptez les pratiques décrites dans cette politique.

2. Données collectées

Données stockées localement sur votre appareil :

  • Nombre de pas et distance parcourue
  • Progression dans les quêtes (checkpoints atteints, XP, badges)
  • Pseudonyme et avatar choisis lors de l'onboarding
  • Préférences d'univers et paramètres de l'application

Ces données sont stockées exclusivement sur votre appareil via AsyncStorage et ne sont transmises à aucun serveur sans votre consentement explicite.

Données collectées avec votre consentement (compte Google optionnel) :

  • Adresse e-mail Google (uniquement pour l'authentification)
  • Identifiant unique de compte
  • Photo de profil (si vous choisissez d'en utiliser une via votre galerie)
  • Données de progression synchronisées avec nos serveurs (Supabase)

3. Permissions utilisées

  • ACTIVITY_RECOGNITION — nécessaire pour comptabiliser vos pas via le capteur du téléphone
  • BODY_SENSORS — accès au podomètre matériel de l'appareil
  • Accès à la galerie photos — uniquement si vous choisissez de personnaliser votre avatar avec une photo
  • Notifications — pour vous alerter lors de l'atteinte d'un checkpoint (avec votre autorisation)

Aucune permission de localisation GPS n'est requise. WalkQuest n'accède jamais à votre position géographique.

4. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Faire progresser votre avatar sur les cartes de quête
  • Sauvegarder votre progression localement
  • Synchroniser votre progression entre vos appareils (si vous avez lié un compte Google)
  • Afficher vos statistiques dans l'application

Nous ne vendons pas, ne louons pas et ne partageons pas vos données personnelles avec des tiers à des fins commerciales.

5. Stockage et sécurité

Les données locales sont stockées via AsyncStorage sur votre appareil Android. Si vous activez la synchronisation cloud, vos données sont transmises de manière sécurisée (HTTPS) vers nos serveurs hébergés sur Supabase (région EU — Irlande), conformément au RGPD.

L'authentification Google est gérée via OAuth 2.0. Nous ne stockons jamais votre mot de passe Google.

6. Conservation des données

Les données locales sont conservées jusqu'à désinstallation de l'application ou suppression via les Paramètres. Les données cloud sont supprimées sur demande via l'option « Supprimer mon compte » dans les Paramètres de l'application.

7. Droits des utilisateurs (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès — consulter vos données dans l'onglet Profil
  • Droit de suppression — supprimer votre compte et toutes vos données via les Paramètres
  • Droit de portabilité — vos données de progression sont stockées dans des formats standards
  • Droit d'opposition — vous pouvez utiliser l'application sans créer de compte

8. Mineurs

WalkQuest est destinée à un public général. Nous ne collectons pas sciemment de données personnelles auprès d'enfants de moins de 13 ans. Si vous avez moins de 13 ans, veuillez utiliser l'application sans créer de compte Google.

9. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. En cas de modification importante, une notification sera affichée dans l'application. La date de dernière mise à jour est indiquée en haut de ce document.

10. Contact

Pour toute question relative à vos données personnelles ou pour exercer vos droits RGPD, utilisez la fonction « Supprimer mon compte » dans les Paramètres de l'application, ou contactez-nous à : josselinsanjuan@gmail.com
